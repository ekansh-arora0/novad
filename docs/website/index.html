1

Velocity Correction

Immediate response. Compares joystick input to actual velocity. If you're being pushed (velocity without input), Novad applies counter-force instantly.

correction = velocityPID(intendedVelocity - actualVelocity)
2

Position Correction

Long-term stability. Returns robot to locked position over time. Only active when position is locked or drift exceeds threshold.

correction = positionPID(lockedPosition - currentPosition)
3

Heading Correction

Rotation lock. Maintains your robot's facing direction. Prevents opponents from spinning you around.

correction = headingPID(lockedHeading - currentHeading)

Add Dependency

    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.novad:novad:1.0.0'
}
2

Configure Robot

NovadConstants.java 
// Set your drivetrain type
public static final DriveType DRIVE_TYPE = DriveType.MECANUM_4_MOTOR;

// Set your odometry type  
public static final OdometryType ODOMETRY_TYPE = OdometryType.THREE_WHEEL;
3

Use in TeleOp

TeleOp.java 
// Initialize (once in init)
Novad novad = new Novad(odometry, drivetrain);

// In your loop - that's it!
while (opModeIsActive()) {
    novad.defense(
        gamepad1.left_stick_x,
        -gamepad1.left_stick_y,
        gamepad1.right_stick_x
    );
}
